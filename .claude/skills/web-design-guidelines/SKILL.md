# Web Design Guidelines

External skill from `vercel-labs/agent-skills` → `web-design-guidelines`

## Purpose

Provides guidelines for creating accessible, responsive, and modern web designs.

## Usage

This skill is automatically referenced by the Homepage Plugin during page and section generation to ensure:
- Accessibility compliance (WCAG 2.1)
- Responsive design patterns
- Modern CSS best practices
- Performance-optimized layouts
